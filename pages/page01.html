Conceptos Generales de un Computador

CONCEPTOS GENERALES

1. Introducción
  • Contexto: Vivimos en la era digital donde la computación impulsa la economía, la educación y la comunicación.
  • Propósito: Entender que una computadora no es solo una máquina, sino una herramienta procesadora de información que transforma datos crudos en resultados útiles.
2. La Computadora
  • Definición: Es un sistema electrónico digital diseñado para recibir datos (entrada), procesarlos mediante algoritmos y generar información (salida).
  • Clasificación
    • Por capacidad: Supercomputadoras (ciencia), Mainframes (bancos), Servidores (redes) y Microcomputadoras (PCs y Laptops).
    • Por movilidad: De escritorio (fijas) y Portátiles (tablets, smartphones, laptops).
3. Componentes de una computadora
  • Hardware
    • Dispositivos de Entrada:Son los que permiten introducir datos al sistema.
      • Ejemplos: Teclado, mouse, micrófono, escáner, cámara web.
    • Dispositivos de Procesamiento y Almacenamiento:El "cerebro" y la "memoria".
      • Procesamiento: El CPU (Procesador) que ejecuta las instrucciones.
      • Almacenamiento: Memoria RAM (temporal/rápida) y Disco Duro/SSD (permanente).
    • Dispositivos de Salida:Los que muestran los resultados al usuario.
      • Ejemplos: Monitor, impresora, altavoces, auriculares.
  • Software
    • Software de Base:El programa que gestiona el hardware y permite que otros programas funcionen.
      • Ejemplo: Windows, macOS, Linux, Android.
    • Software de Aplicación:Programas diseñados para tareas específicas del usuario.
      • Ejemplo: Word, Chrome, WhatsApp, Photoshop, videojuegos.
    • Software de Programación:Herramientas que usan los desarrolladores para crear otros programas.
      • Ejemplo: Visual Studio Code, Python, Java.
4. Licencias de Software
  • Software Libre:Da libertad al usuario para usarlo, estudiarlo, modificarlo y distribuirlo. No siempre es gratis, pero su código es abierto
    • Ejemplo: Sistema Operativo Linux, navegador Firefox.
  • Software Comercial:Pertenece a una empresa y el usuario paga por una licencia de uso. No se puede modificar ni repartir legalmente.
    • Ejemplo: Microsoft Office, Windows, Adobe Creative Cloud
